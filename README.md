# TravelTails UK

A responsive web application connecting pet owners with verified pet-friendly 
accommodations across the United Kingdom.

## Version History

### v1.0 - Academic Submission (November 2025)
Educational milestone for Web Technologies 2 coursework at St George's, 
University of London.

**Core Features:**
- Responsive design with mobile-first approach
- Three validated forms (registration, contact, trip booking)
- Accessible navigation with SlickNav
- Semantic HTML5 structure
- WCAG 2.1 compliant accessibility
